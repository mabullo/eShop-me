== Description ==

Plugin for the rapid administration of the eshop's product.

The plugin is a tool for eShop.
Shows only the categories that have items inside and eliminating the hierarchy (so you do not have duplicate elements);
clicking on a category displays all its products.
For every products you can modify or add option, price ecc..


== Installation ==

Download the plugin, upload to your Wordpress plugins directory and activate.
Under the post menù you can find eshopme submenù.
